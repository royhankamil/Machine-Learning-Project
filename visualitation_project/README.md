# Dashboard Project Achmad Royhan Kamil

## Requirements
```
Sebelum menjalankan dashboard, pastikan Anda telah menginstal hal-hal berikut::

- Python 3.x
- Streamlit
- seaborn
- matplotlib
- numpy
- pandas
```

## Setup Environment 
```
cd Dashboard
pipenv install
pipenv shell
pip install -r requirements.txt
```
## Run steamlit app
```
streamlit run dashboard.py
```
